wp-query用法

WP_Query 类是 WordPress 文章查询的核心，它的文章查询部分提供了很多可用参数，经过好几个版本的升级完善，让它越来越强大，本文就可以让你详细的了解所有查询参数和技巧。

作者

查询和某些作者（用户）有关的文章，可以使用 4 个参数：

  • author（整数）：用户 ID
  • author_name（字符串）：用户的昵称（“user_nicename” 字段）
  • author__in（数组）：用户 ID
  • author__not_in（数组）：用户 ID
